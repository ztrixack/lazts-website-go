Technology Stack

สำหรับ Website ก่อร่างสร้างตัวขึ้นมาจากเทคโนโลยีนี้

  • Go: Backend server สำหรับ serve ข้อมูลต่างๆที่ render แล้วออกมาให้
  • HTMX: JavaScript Action
  • TailwindCSS: จัดการ CSS บน class

  • Cloudflare: หน้าด่านสำหรับทำ SSL/Cache/Domain Mapping

History

เริ่มแรกเดิมที เราเขียนบทความมาตั้งแต่ตอนเรียนอยู่ปี 2 แล้ว ในตอนนั้นจะเป็นการเขียนสรุปการเรียนต่างๆใส่ใน Google Doc เอาไว้สำหรับเตรียมสอบและอยากรวบรวมข้อมูลและความรู้ที่เราได้รับมาบันทึกไว้ที่ไหนซักที่ กะจะเอาไว้อ่านเอง ไม่ได้มีความคิดที่จะเปิดแบบ public หรือมีเว็บไซด์ เพราะยังทำไม่เป็น จนกระทั่งเริ่มทำงานไปได้ซักพัก

หลังจากทำงานได้จนอยู่ตัวแล้ว เวลาว่างมากขึ้น ก็มีความคิดที่ว่าเราน่าจะลองแชร์ข้อมูลที่เรามีเผยแพร่ออกไปบ้าง เผื่อจะมีค่าขนมเล็กๆ น้อยๆ แต่ในช่วงเวลานั้นก็ยังไม่มีความรู้ทางด้านเว็บไซด์มากนัก เราพึ่งพาเว็บสำเร็จรูปอย่าง Google Site, Blogger หรือ Wordpress แต่ก็ยังไม่ค่อยถูกใจเท่าไร เพราะมันไม่สามารถปรับแต่งได้รูปแบบที่เราต้องการ อย่างพวกสมการคณิตศาสตร์ รูปภาพ หรือการจัดกลุ่มต่างๆ

พอเขียนไปได้ซักพัก ก็เริ่มศึกษาการเขียนเว็บไซด์ และก็ลองเอาการเรียนการเว็บไซด์นี้แหละมาเขียนบทความลงเว็บอีกที ประโยชน์​ 2 ต่อ คุ้มค่า! และเพื่อที่จะได้ปรับเปลี่ยนให้ตรงกับสิ่งที่เราอยากได้มากขึ้น จึงได้เริ่มลองเขียน Frontend ขึ้นมาก่อนจาก JavaScript Framework สุดฮิตจากค่าย Meta (หรือตอนนั้นก็คือ Facebook) React โดยมี Backend เป็น Firebase RealTime Database ไว้สำหรับเก็บข้อมูล และพยายามย้ายบ้านมาลงที่นี่ทั้งหมด

ทุกอย่างดูเหมือนจะเป็นไปได้ด้วยดี เพราะ React ก็สามารถรังสรรค์ได้ตามที่เราต้องการ มีระบบ login วิธีการเขียนบทความผ่านหน้าเว็บได้เลย พร้อมทั้งยังได้ฝึกการเขียนเว็บอีกด้วย แต่พอเอาเข้าจริงก็กลายเป็นว่าติดเขียนเว็บ ไม่ได้เขียนบทความเลย ก็ปรับโน่น แต่งนี่ไปเรื่อย

ส่วนที่กินเวลามากที่สุดก็คือการทำ markdown ผ่านหน้าเว็บและการจัดเก็บข้อมูล มันนานจดเริ่มรู้สึกได้ว่ามาผิดทางแล้ว เราควรจะเน้นไปที่ข้อมูลหรือความรู้ที่เราจะเขียนลงไปมากกว่ามานั่งแก้ปัญหาจุกจิกนี้ จึงเริ่มกลับมองหา platform สำเร็จรูปสำหรับเขียนบทความอย่าง Ghost จึงเริ่มการทำเว็บไซต์ในยุคที่ 2

React + Ghost ค่อนข้างตอบโจทย์มากกับการที่ Ghost มีระบบจัดการที่ค่อยข้างดี แม้จะมีจุดขัดใจเล็กๆน้อยๆ บ้าง ก็ยอมๆ กันไป เราจึงอยู่กับ React + Ghost เป็นระยะเวลาที่นานพอสมควร แต่ด้วยการร้างการเขียนไปนานและงายยุ่งมากๆ ก็ไม่ค่อยได้เข้าไปใช้ กอปรกับค่า server ก็เริ่มกินค่าใช้จ่ายไปมากขึ้นเรื่อยๆ จึงตัดสินใจปิดตัว server ลงไป และเอาตังไปหาอะไรอร่อยๆ กินแทน

หลังจากนั้นก็ได้มารู้จักกับ Notion ซึ่งเป็น application จดโน๊ตที่ใช้ได้ฟรี รองรับการทำงานผ่าน web browser และสามารถเขียนบทความข้อมูลต่างๆ ที่เราอยากจดไว้ได้ง่ายมากๆ จึงได้เริ่มลองใช้และใช้มาจนถึงปัจจุบัน

เรื่องเหมือนจะจบ แต่ก็ไม่ พอเราเขียนเว็บเป็น เราก็อยากมีที่ลองอะไรใหม่ๆ อย่าง Svelte หรือ Vue ซึ่งก็เริ่มลองเขียนดู โดยทำเป็นแบบ static file และอัพขึ้น Google Firebase Hosting ซึ่งใช้งานได้ฟรีๆ แต่ก็็มาติดกับดักเรื่องเวลาว่างและปัญหา technical อื่นๆ อยู่ดี จึงต้องปิดตัวลงไปก่อน

จนสุดท้ายที่ทำงานเริ่มอยู่ตัวและว่างมากขึ้น ก็คิดว่าเราน่าจะลองกลับมาเขียนบทความแชร์อีกครั้ง โดยนำบทความทั้งหลายที่เคยเขียนตั้งแต่อดีตมาปัดฝุ่นใหม่และอัพเดทในทันเทคโนโลโลยีมากขึ้น โดยไม่จำเป็นต้องทำอะไรเวอร์วังอลังการแบบตอนแรก (มั้ง) และจะใช้สิ่งที่เราถนัดในตอนนั้นนั่นคือใช้ภาษา Go เป็น Backend และจัดการทุกๆ อย่างให้เสร็จสรรพด้วยการ code เลย และใช้ HTMX มาจัดการกับ Frontend แทน โดย deploy ขึ้น Google Cloud Run ซึ่งใช้ได้ฟรีถ้าไม่เกินโควต้า
