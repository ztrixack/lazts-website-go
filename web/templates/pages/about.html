History

เริ่มแรกเดิมทีเขียนบทความมาตั้งแต่ปี 2016 ในช่วงเวลานั้นยังไม่มีความรู้ทางด้านเว็บไซด์มากนัก แต่ก็อยากรวบรวมข้อมูล ความรู้ที่เราได้รับรู้มา บันทึกไว้ซักที่ จึงได้เริ่มเขียนบทความขึ้น

ในช่วงแรกๆ (ปี 2016 - 2017) จะยังพึ่งพาเว็บสำเร็จรูปอย่าง Google Site, Blogger, หรือ Wordpress แต่ก็ยังไม่ค่อยถูกใจเท่าไร เพราะมันไม่ได้รูปแบบที่เราต้องการ อย่างพวกสมการคณิตศาสตร์ รูปภาพ หรือการจัดกลุ่มบทความ จึงเริ่มศึกษาการเขียน website เพื่อที่จะได้ปรับเปลี่ยนให้ตรงกับสิ่งที่เราอยากได้มากขึ้น ในปีถัดมา จึงได้เริ่มลองเขียน Frontend ขึ้นจาก React โดยมี Backend เป็น Firebase RealTime Database และพยายามย้ายบ้านมาลงที่นี่ทั้งหมด

ทุกอย่างดูเหมือนจะเป็นไปได้ด้วยดี เพราะ React ก็สามารถรังสรรค์ได้ตามที่เราต้องการ มีระบบ login วิธีการเขียนบทความผ่านหน้าเว็บได้เลย พร้อมทั้งยังได้ฝึกการเขียนเว็บอีกด้วย แต่พอเอาไปใช้งานจริงก็จะติดปัญหาทาง Technical หลายอย่าง และมีเรื่แงจุกจิกมากซะจนไม่ได้เขียนบทความเลย เพราะมัวแต่ปรับนั่น แต่งโน่นไปเรื่อย โดยเฉพาะการเขียนบทความผ่านหน้าเว็บ

ก็ทำเว็บสำหรับบทความของตัวเองมาเรื่อยๆ ซึ่งทำมากซะจนไม่มีเวลาเอาไปหาข้อมูลเพื่อเขียนบทความใหม่ๆ เลย ตอนนั้นก็เริ่มท้อแท้เพราะมันไม่เสร็จซักที จึงเริ่มกลับมองหา platform สำเร็จรูปอย่าง medium และหลังบ้านสำเร็จรูปสำหรับเขียนบทความอย่าง Ghost

สำหรับ medium มันค่อนข้างตรงกับสิ่งที่เราต้องการมาก แต่หลังจากที่เขียน backend เป็นแล้ว เราก็อยากได้การทำ backend application ขึ้นมาแปะบนหน้าเว็บด้วย ดังนั้นจะเหลือเพียง 1 ตัวเลือกนั่นก็คือ ใช้ React เป็น frontend และให้ Ghost เป็น backend ไป

React + Ghost ค่อนข้างตอบโจทย์ จึงใช้เป็นระยะเวลาที่นานพอสมควร แต่ cost ก็ได้โตตามระยะเวลาไปด้วย จึงเริ่มเปลี่ยนแผนในการออกบทความเป็นการแปลงข้อมูลทั้งหมดมาเป็น static HTML เพราให้โค้ดสามารถลง Firebase Hosting ซึ่งใช้ฟรีได้
