Technology Stack

สำหรับเว็บไซต์นี้ เราใช้เทคโนโลยีหลายอย่างเพื่อให้ได้ผลลัพธ์ที่ตรงกับความต้องการและสามารถจัดการได้ง่าย โดยเทคโนโลยีที่ใช้มีดังนี้:

  • Go: ใช้เป็น Backend server สำหรับการเสิร์ฟข้อมูลที่ถูก render แล้วออกมาให้กับผู้ใช้งาน
  • Markdown: ใช้สำหรับเขียนบทความ ทำให้การจัดการเนื้อหาเป็นไปได้อย่างสะดวกและมีโครงสร้างที่ดี
  • HTMX: ใช้ในการจัดการ JavaScript Action ช่วยให้หน้าเว็บมีความโต้ตอบและประสบการณ์การใช้งานที่ดีขึ้น
  • TailwindCSS: ใช้ในการจัดการ CSS บน class ทำให้การออกแบบและปรับแต่งหน้าเว็บเป็นไปอย่างรวดเร็วและง่ายดาย
  • Google Cloud Run: ใช้สำหรับการ deploy server(less) ทำให้สามารถขยายขนาดการใช้งานได้ตามต้องการโดยไม่ต้องจัดการ server เอง
  • Cloudflare: ทำหน้าที่เป็นหน้าด่านสำหรับ SSL, Cache และ Domain Mapping เพื่อเพิ่มความปลอดภัยและประสิทธิภาพในการเข้าถึงเว็บไซต์

กว่าจะเป็นเว็บไซต์นี้

เริ่มแรกเดิมที เราเริ่มเขียนบทความตั้งแต่ตอนเรียนอยู่ปี 2 โดยใช้ Google Doc สรุปเนื้อหาการเรียนเพื่อเตรียมสอบและเก็บความรู้ที่ได้รับมา ตอนนั้นยังไม่ได้คิดจะเผยแพร่หรือสร้างเว็บไซต์เพราะยังไม่มีความรู้ด้านนี้ จนกระทั่งเริ่มทำงานได้ซักพัก

เมื่อเริ่มทำงานจนอยู่ตัว เวลาว่างมากขึ้นก็คิดว่าเราน่าจะแชร์ข้อมูลที่มีให้คนอื่นๆ เผื่อจะได้ค่าขนมเล็กๆ น้อยๆ ตอนนั้นยังไม่มีความรู้ในการสร้างเว็บไซต์ จึงพึ่งพาเว็บสำเร็จรูปอย่าง Google Site, Blogger หรือ WordPress แต่ก็ไม่ค่อยพอใจเพราะปรับแต่งไม่ได้ตามต้องการ เช่น การแสดงสมการคณิตศาสตร์ รูปภาพ หรือการจัดกลุ่มข้อมูลต่างๆ

พอเขียนไปได้ซักพัก ก็เริ่มศึกษาการเขียนเว็บไซต์ และลองนำความรู้มาเขียนบทความ ได้ประโยชน์สองต่อ! เราเริ่มต้นจากการเขียน Frontend ด้วย React (จาก Meta หรือ Facebook) โดยใช้ Firebase RealTime Database เป็น Backend สำหรับเก็บข้อมูล และพยายามย้ายบทความมาลงที่นี่ทั้งหมด

ทุกอย่างดูเหมือนจะไปได้ด้วยดี เพราะ React ทำให้สามารถสร้างสิ่งที่ต้องการได้ มีระบบ login และสามารถเขียนบทความผ่านหน้าเว็บได้ แต่ก็กลายเป็นว่าติดการเขียนเว็บมากกว่าเขียนบทความ ปรับนั่นแต่งนี่ไปเรื่อย

ส่วนที่กินเวลามากที่สุดคือการทำ markdown ผ่านหน้าเว็บและการจัดเก็บข้อมูลที่ไม่ตรงใจซักที จนเริ่มรู้สึกว่ามาผิดทาง เราควรเน้นที่เนื้อหามากกว่าการแก้ปัญหาจุกจิก จึงเริ่มมองหา platform สำเร็จรูปสำหรับเขียนบทความอย่าง Ghost และเริ่มทำเว็บไซต์ในยุคที่สอง

React + Ghost ตอบโจทย์มากเพราะ Ghost มีระบบจัดการที่ดี แม้จะมีจุดขัดใจเล็กๆ น้อยๆ แต่ก็ยอมรับได้ เราจึงใช้ React + Ghost อยู่ระยะเวลานานพอสมควร แต่ด้วยการร้างการเขียนไปนานและงานยุ่ง ค่า server เริ่มเพิ่มขึ้นเรื่อยๆ จึงตัดสินใจปิดตัว server และนำเงินไปหาอะไรอร่อยๆ กินแทน

หลังจากนั้น เราได้รู้จักกับ Notion ซึ่งเป็น application จดโน๊ตที่ใช้งานฟรี รองรับการทำงานผ่านเว็บบราวเซอร์ และสามารถเขียนบทความได้ง่ายมากๆ จึงเริ่มลองใช้และใช้มาจนถึงปัจจุบัน ซึ่งก็ตอบโจทย์ที่ต้องการมากๆ

เรื่องเหมือนจะจบ แต่ก็ไม่ เพราะพอเราเขียนเว็บเป็น เราก็อยากลองอะไรใหม่ๆ บ้าง อย่างพวก Svelte หรือ Vue โดยลองทำเว็บเป็น static file และอัพขึ้น Google Firebase Hosting ซึ่งใช้งานฟรี แต่ทำไปซักพักก็ติดปัญหาเวลาว่างและปัญหา technical อื่นๆ ที่ทำยังไงก็ยังไม่ถูกใจซักที (เอาใจยากจริง!) จึงต้องปิดตัวลงไปก่อนอีกครั้ง

สุดท้าย เมื่อทำงานจนอยู่ตัวและมีเวลาว่างมากขึ้น ก็คิดว่าเราน่าจะกลับมาเขียนบทความแชร์อีกครั้ง นำบทความที่เคยเขียนมาปัดฝุ่นและอัพเดทให้ทันสมัย โดยไม่ต้องทำอะไรซับซ้อน ใช้ภาษา Go เป็น Backend และจัดการทุกอย่างด้วย code ใช้ HTMX สำหรับ Frontend และ deploy ขึ้น Google Cloud Run ที่ใช้งานฟรีถ้าไม่เกินโควต้า

จนในที่สุดก็พยายามที่จะเขียนเว็บออกมา โดยลงแรงกับการทำหน้าเว็บให้น้อยที่สุด โดยที่ยังมี features ที่อยากได้ทั้งหมดออกมา จนกลายมาเป็นเว็บไซต์นี้ขึ้นมา

สุดท้าย ขอบคุณทุกๆ ท่านที่เข้ามาอ่าน หากมีข้อผิดพลาดหรือข้อเสนอแนะ สามารถ comment คุยกันได้ หวังว่าเว็บไซต์นี้จะมีประโยชน์แก่ผู้ที่เข้ามาเยี่ยมชมบ้างนะครับ 😁
